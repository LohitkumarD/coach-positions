# Pilot and Tuning Guide

## Pilot Scope
- 3-5 high-traffic stations
- 7 operational days minimum
- Mix of peak/off-peak shifts

## Data to Capture
- Submission count per train
- Decision confidence distribution
- Number of conflict cases
- Alert acknowledgement time
- Announcement correction count

## Weekly Calibration
- Tune source weights
- Tune high/medium/low thresholds
- Tune proximity bonus caps
- Tune reliability reward/penalty rates

## Exit Criteria for Zone Rollout
- >= 85% trains have medium/high confidence before ETA
- Correction rate reduced by >= 40%
- Alert acknowledgement median <= 2 minutes
- Supervisor override rate stable and declining
